HTML tag examples

<!-- "Comment Here" --> - leave a single or multi line comment in a HTML document.

example: <!--some important comment-->

example: <!--some important comment
continuation of comment-->

<!DOCTYPE> Tells the browser this is a HTML5 document.

example: place as very first tag in a HTML file

<a> put a hyperlink in this tag.

example: <a href='https://www.google.com/'>click me!</a>

looks like: click me!

<abbr> Defines an abbreviation or an acronym.

example: <abbr title="Hypertext Markup Language">HTML</abbr>

looks like: HTML

<address> Defines name, email and/or phone number for the author/owner of a document.

example:
<address> Authored by Jon Doe.
Find us at:
example.com
<address>

          <map name="appleorange">
            <area shape="circle" coords="51,58,25" href="https://www.google.com" alt="Apple">

          </map>
